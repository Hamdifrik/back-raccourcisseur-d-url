# Backend - URL Shortener API

## Description

Ceci est la partie backend de l'application de raccourcissement d'URL. Elle est construite avec Node.js, Express et MongoDB. Elle fournit une API qui permet de raccourcir une URL.

## Prérequis

- Node.js installé sur votre machine
- MongoDB installé (ou un cluster MongoDB en ligne, comme MongoDB Atlas)

## Installation

1. Clonez le dépôt du backend :

   ```bash
   git clone https://github.com/Hamdifrik/back-raccourcisseur-d-url
   cd back-raccourcisseur-d-url
   npm install
   node server.js
